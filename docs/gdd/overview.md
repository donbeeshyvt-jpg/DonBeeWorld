# 冬蜜大陸世界 GDD 摘要

> 版本：0.1.0  
> 更新日期：2025-11-12

## 1. 世界設定

- **主舞台：史萊姆酒館** – 位於異世界交界的中立安全區，所有玩家的主要互動、聊天、交易都在此進行。
- **延伸場景**
  - **地下工作室**：冬蜜的私人空間，用於加工、煉金、武器強化。
  - **戰亂居酒屋**：高危區域，提供高階冒險與急救任務，報酬極高。
  - **帳篷營地**：友善新手與天氣事件中心，負責補給與生活用品。
  - **次元裂縫**：遠方的終極挑戰，後續模組可在此引入 Boss 或季節副本。

## 2. 核心玩法模組

| 模組 | 說明 | 互動 |
| --- | --- | --- |
| 採集／採礦／釣魚／採藥 | 參考 Melvor Idle 的多技能設計，每項技能都有進度條、成功率、離線收益 | 生成素材、技能經驗、成就計數 |
| 加工／煉金／烹飪 | 將素材轉換為裝備或高階商品，配方可透過冒險或任務取得 | 產出將進入市場或任務獎勵 |
| 冒險派遣與戰鬥 | 玩家可派遣隊伍至不同區域；支援主動戰鬥、Boss、副本與比武 | 套用裝備、技能、寵物加成，並可在 Discord 公布戰報 |
| 任務／成就／活動 | 支援每日／每週／主線任務，伺服器可下發季節活動與成就 | 任務完成後可更新公告、跑馬燈 |
| 市場（股市） | 商品每日或自訂週期變動價格；玩家可自由掛單或即時交易 | 後台可調波動參數與限時物品 |
| 雙貨幣經濟 | `金幣` 由遊戲內獲得；`魂幣` 透過贊助取得，並可自訂匯率 | 商城支持魂幣付款、金幣找零 |
| 社交／聊天／Discord | 遊戲內置頂聊天 + Discord Bot 雙向同步；支援 DC 樣式語法 | GM 可排程公告、跑馬燈與活動提醒 |

## 3. 玩家成長

- **角色屬性**：生命、戰鬥力、魔力、防禦、速度。
- **技能樹**：採集、戰鬥、商業三大領域，每個領域有專屬天賦與被動效果。
- **裝備欄位**：武器、護甲、飾品、工具、消耗品；裝備決定戰鬥與採集效率。
- **寵物／夥伴**：提供被動增益或特殊技能，獲得途徑包括冒險、活動、商店。

## 4. 貨幣與經濟

- 預設匯率：`1 魂幣 = 100 金幣`（可在後台隨時調整）。
- 贊助入口可購買魂幣、限定外觀或季票；魂幣消費時若需找零以金幣計算。
- 市場價格波動、任務獎勵、贊助活動都會影響整體經濟；需定期監控並調整。

## 5. 內容製作流程

1. 於 Google Sheet 依 `docs/data/specs/` 提供的格式維護內容（任務、採集點、配方、裝備、寵物等）。
2. 利用後台匯入／匯出功能同步資料庫，保留版本記錄與審核流程。
3. 新活動或季節任務需同時準備公告、跑馬燈、Discord 預排訊息與市場調整。

## 6. 後續擴充

- **次元裂縫副本**：多階段 Boss、世界線分支、聯盟挑戰。
- **公會 / 小隊系統**：支援玩家自組隊伍、共享倉庫、對戰。
- **跨平台同步**：行動版／桌面版共用帳號與雲端儲存。

> 完整 GDD 將於後續章節補充（技能樹細節、活動清單、故事腳本等）。本文件作為初始方向與資料庫設計的依據。***
